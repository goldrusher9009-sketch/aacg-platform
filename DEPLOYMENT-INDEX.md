## Essential Commands

### Deploy to Railway
```bash
# 1. Push to GitHub
git push origin main

# 2. Set environment variables in Railway.app dashboard
# (See RAILWAY-DEPLOYMENT-GUIDE.md for details)

# 3. Deployment happens automatically
# Monitor at: https://railway.app/dashboard
```

### Test After Deployment
```bash
# Health check
curl https://YOUR-APP.railway.app/api/health

# Login endpoint
curl -X POST https://YOUR-APP.railway.app/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"password"}'

# Mechanics liens
curl https://YOUR-APP.railway.app/api/mechanics-liens

# Photo analysis
curl https://YOUR-APP.railway.app/api/photo-analysis
```

### View Logs
```bash
# In Railway.app dashboard:
# 1. Click "Logs" tab
# 2. Select "app" service
# 3. View real-time logs
# 4. Filter by severity (INFO, WARN, ERROR)
```
