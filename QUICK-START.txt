================================================================================
  COMMON ISSUES & FIXES
================================================================================

"bash: command not found"
  → Install Git for Windows: https://git-scm.com/download/win
  → Then try the script again

"Permission denied"
  → Run: chmod +x push-to-github.sh
  → Then: ./push-to-github.sh

"fatal: Authentication failed"
  → Your Personal Access Token is wrong or expired
  → Create a new one: https://github.com/settings/tokens
  → Try the script again

"Credentials not configured" error in portal
  → Follow the "Configure Supabase Credentials" section above
  → Make sure to use YOUR actual Supabase URL and key
  → Refresh the page

Portal doesn't load
  → Check GitHub Pages is enabled: go to Settings → Pages
  → Verify branch is "main" and folder is "/"
  → Wait a minute and refresh again
