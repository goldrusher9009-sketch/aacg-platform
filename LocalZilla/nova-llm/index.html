Why Nova?

Training a frontier AI model costs hundreds of millions of dollars. Nova proves there's another way — by combining the spare GPU cycles of thousands of volunteers into a single training run.

⚡

Novel Training Algorithm

Adaptive Swarm Optimization (ASO) reduces bandwidth by 50-200x while protecting against malicious participants via trust-weighted aggregation.

🌐

Train From Anywhere

Any GPU works — NVIDIA, AMD, Apple Silicon, even CPU. One command to join. Leave anytime without breaking anything.

🔒

Byzantine Fault Tolerant

Trust scoring automatically detects and downweights bad actors. Even with 30-40% malicious workers, training converges correctly.

📚

Smart Data Pipeline

Quality scoring, deduplication, and curriculum learning ensure the model trains on the right data in the right order.

🚀

Adaptive Communication

Workers learn when to sync — no fixed schedule. Less bandwidth wasted, faster convergence on slow connections.

🤝

100% Open Source

Every line of code — model, training, tokenizer, infrastructure — is open. MIT licensed. No strings attached.

Join the Swarm in 60 Seconds

If you have a computer with a GPU (or even just a CPU), you can contribute to training Nova.

# 1. Install dependencies pip install torch numpy websockets # 2. Clone the repo git clone https://github.com/YOUR_USERNAME/nova.git cd nova # 3. Join the swarm python -m nova.swarm.worker --coordinator wss://nova-coordinator.example.com
1

Install

Clone the repo and install PyTorch. Takes under a minute.

2

Connect

Run the worker script. It automatically connects to the coordinator and downloads the latest model.

3

Train

Your GPU starts training locally. Compressed updates are sent to the swarm periodically. Leave it running as long as you want.

4

Earn Trust

The more you contribute, the higher your trust score. Top contributors are featured on the leaderboard.

How ASO Works

Adaptive Swarm Optimization is a new algorithm designed for unreliable internet connections.

Feature Standard Training Hivemind Nova ASO
Communication Every step Fixed interval Adaptive (learned)
Compression None (1x) ~10x 50-200x (SVD + error feedback)
Malicious protection None None Trust-weighted aggregation
Worker dropout Crash Retry Seamless (trust adjusts)
Data strategy Random shuffle Random shuffle Curriculum + quality scoring
